# 6x6 Sudoku Game

This is a **6x6 Sudoku game** built with Python and Tkinter. The game generates a random Sudoku board and allows you to fill in the missing numbers. You can check your solution or start a new game anytime.

## Features

- Generates a random 6x6 Sudoku board.
- Allows you to fill empty cells.
- Checks if your solution is correct.
- Starts a new game with a different puzzle.


## How to Use

1. **Run the program.** Open a terminal and navigate to the folder where the file is located. Then run the Python file: **python sudoku.py**

2. **Fill the empty cells.** Click on a cell and type a number.
3. **Check your solution.** Press the **Check** button, the program will tell you if: 

- Your Sudoku is complete.
- There are duplicate numbers in a row, column, or block.
- Your solution is correct.
4. **Start a new game.** Press the **New Game** button to generate a new puzzle. All previous inputs will be cleared.
